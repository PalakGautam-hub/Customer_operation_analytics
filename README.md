# Customer Operations Analytics System

> **Palak Gautam** — Analytics Dashboard tracking Retention, Revenue Risk & SLA Performance

---

## System Architecture

```
Raw Customer Data (customer_operations.csv)
        ↓
Python ETL Pipeline (pipeline.py)
  • Cleans & validates data
  • Computes KPIs
  • Exports processed CSV + JSON
        ↓
Chart.js Web Dashboard (index.html)
  • Loads CSV via fetch()
  • Retention trend line chart
  • Revenue risk bar chart
  • SLA compliance doughnut
        ↓
Power BI Executive Dashboard
  • Imports processed_customer_operations.csv
  • KPI cards, slicers, trend visuals
```

## Dataset Columns

| Column | Description |
|--------|-------------|
| `customer_id` | Unique customer identifier |
| `month` | Time dimension (Jan–Jun) |
| `revenue` | Monthly revenue from customer |
| `retained` | Churn indicator (Yes/No) |
| `sla_met` | SLA compliance (Yes/No) |
| `risk_level` | Revenue risk tier (Low/Medium/High) |
| `tickets_open` | Support tickets raised |
| `tickets_resolved` | Support tickets closed |

## Step 1 — Run the Python Pipeline

```bash
cd python_pipeline
pip install pandas
python pipeline.py
```

**Outputs:**
- `data/processed_customer_operations.csv` — enriched with `retention_flag`, `sla_flag`, `ticket_resolution_rate`
- `data/kpi_summary.json` — overall KPI snapshot
- `data/monthly_summary.json` — month-by-month aggregates

## Step 2 — Web Dashboard

Serve locally (required for CSV fetch):

```bash
# Python
python -m http.server 8000
# then open http://localhost:8000
```

Or open `index.html` directly — static fallback data loads automatically.

## Step 3 — Power BI Dashboard

1. Open **Power BI Desktop**
2. **Get Data → Text/CSV** → import `processed_customer_operations.csv`
3. Create these **DAX Measures**:

```dax
Retention Rate =
AVERAGE(customer_operations[retention_flag])

SLA Compliance =
AVERAGE(customer_operations[sla_flag])

Revenue at Risk =
CALCULATE(
    SUM(customer_operations[revenue]),
    customer_operations[risk_level] = "High"
)

Total Revenue =
SUM(customer_operations[revenue])
```

4. Build these visuals:

| Visual | Type | Axis / Legend | Values |
|--------|------|---------------|--------|
| Retention Trend | Line Chart | `month` | `Retention Rate` |
| Risk Distribution | Bar Chart | `risk_level` | `revenue` |
| SLA Performance | Donut Chart | `sla_met` | `customer_id` (Count) |
| KPI Cards | Card | — | All 4 measures |

5. Add **Slicers**: `month`, `risk_level`, `retained`

## Step 4 — Automation (Windows Task Scheduler)

Schedule pipeline to run daily:

```
Program: python
Arguments: C:\path\to\python_pipeline\pipeline.py
Trigger: Daily at 06:00
```

Power BI dataset refresh reads the updated CSV automatically.
