## ✨ Features

- 📈 **Retention Trend** — Month-over-month customer retention line chart
- 💰 **Revenue Risk Distribution** — Bar chart segmenting customers by risk tier (Low / Medium / High)
- ✅ **SLA Compliance** — Donut chart showing met vs missed SLA targets
- 🧮 **Animated KPI Cards** — Live-calculated retention rate, high-risk accounts, SLA %, and automation gain
- 📤 **Export Button** — Downloads KPI snapshot as CSV with one click
- 🔄 **Python Pipeline** — Cleans data, computes KPIs, exports processed dataset for Power BI
- 📊 **Power BI Ready** — DAX measures and dashboard layout documented

## 📦 Dataset Schema

`data/customer_operations.csv`

| Column | Type | Description |
|--------|------|-------------|
| `customer_id` | int | Unique customer identifier |
| `month` | string | Time dimension (Jan–Jun) |
| `revenue` | float | Monthly revenue from customer |
| `retained` | Yes/No | Churn indicator |
| `sla_met` | Yes/No | SLA compliance |
| `risk_level` | Low/Medium/High | Revenue risk tier |
| `tickets_open` | int | Support tickets raised |
| `tickets_resolved` | int | Support tickets closed |

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/PalakGautam-hub/Customer_operation_analytics.git
cd Customer_operation_analytics
```

### 2. Run the Python Pipeline

```bash
pip install pandas
cd python_pipeline
python pipeline.py
```

## 📊 Power BI Dashboard

### Import Data
1. Open **Power BI Desktop**
2. **Get Data → Text/CSV** → select `data/processed_customer_operations.csv`

### DAX Measures

```dax
Retention Rate =
AVERAGE(customer_operations[retention_flag])

SLA Compliance =
AVERAGE(customer_operations[sla_flag])

Revenue at Risk =
CALCULATE(
    SUM(customer_operations[revenue]),
    customer_operations[risk_level] = "High"
)

Total Revenue =
SUM(customer_operations[revenue])
```

### Dashboard Layout

| Visual | Type | Axis | Values |
|--------|------|------|--------|
| Retention Trend | Line Chart | `month` | `Retention Rate` |
| Risk Distribution | Bar Chart | `risk_level` | `revenue` |
| SLA Performance | Donut Chart | `sla_met` | Count of `customer_id` |
| KPI Summary | Card ×4 | — | All 4 measures |

**Add slicers:** `month`, `risk_level`, `retained`

### Automate with Windows Task Scheduler

```
Program:   python
Arguments: C:\path\to\python_pipeline\pipeline.py
Trigger:   Daily at 06:00
```
Power BI refresh reads the updated CSV automatically → **real-time stakeholder visibility.**
